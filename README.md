>
> Nei, spennende!

# Henvendelser
Spørsmål knyttet til koden eller prosjektet kan stilles som issues her på GitHub.

## Hvorfor heter det Spennende?

Spennende er ansvarlig for å håndtere situasjoner der det har skjedd endringer på en person i Infotrygd. Da sendes det et signal over Kafka fra stormaskin-miljøet, og dette bruker vi til å kjøre oppdateringer av våre personer i Spleis.

Alt dette er litt spennende, for det er mye som kan gå galt, og mye som har gått galt.

## For NAV-ansatte
Interne henvendelser kan sendes via Slack i kanalen #område-helse.
